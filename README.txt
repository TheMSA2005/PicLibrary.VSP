Read This Website before Opening the Webpage --------------------------------

Project Title: PicLibrary.VSP

Purpose: This is a static (mostly) website which contains a gallery of photos 
on multiple topics which user can use to create themes, brochures or wallpaper. 
More topics and photos will be added in the future. 
For users to give their reviews and opinion we have gave them phone number, email and also form.

Technologies: HTML, CSS , JavaScript and GoogleScript 
HTML: for building structure
CSS: for designing 
JavaScript: for download & dark/light switch button (interactibility)
JavaScript + Google Script: Since php. and js. are advanced according to our level. We decided to use google script. So , responses of the form would be collected on Google Sheets. The code were taken from online repositories or YouTube Tutorials.

Structure: This Website has primarily three pages 
*Home
*Explore (with 8 Topics)
*Contact

Home: In Home, there are two sections: *Welcome *Purpose
Welcome: to welcome users	Purpose: To let new users know how to run the website.

Explore: This is the soul of the website as when you go there you can select your desired topic and then view or download picture.

Contact: Also contain two sections: *About-Me *Contact
About-Me: It is a sort of portfolio.
Contact: To give reviews and feedbacks.

Why two JS File?
Ans: In Contact section we needed Form Collection but it was not needed in other pages so we made two js file one with form collection activity and one without it. So, website don't take a lot of time to load JS Code.
